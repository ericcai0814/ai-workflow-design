# 精選 AI 協作對話

高品質 AI 協作對話的精選範例，展示有效的 Prompt 模式和問題解決策略。

## 使用指引

每個精選對話包含：

1. **背景說明**：任務目標和情境
2. **關鍵對話**：展示有效的 Prompt 模式
3. **學習重點**：可複用的技巧標註

### 對話格式

```markdown
> **User**: 用戶的問題或指令

> **AI**: AI 的回應
```

### 如何使用這些範例

- **學習 Prompt 技巧**：觀察用戶如何提出有效的問題
- **理解協作模式**：了解 AI 如何逐步解決問題
- **複用模式**：將學習重點應用到自己的工作中
